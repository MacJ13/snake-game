# Snake game

## Description

This is simple snake game.

## Link

[Snake Game live demo](https://macj13.github.io/snake-game/)

## Technologies

- html
- css
- typescript
- canvas api
- vite

## Setup

to run this project, install it locally using git and npm:

```

$ git clone https://github.com/MacJ13/snake-game.git
$ cd snake-game
$ npm install
$ npm run dev

```
